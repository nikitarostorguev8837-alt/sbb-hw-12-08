# Домашнее задание к занятию "`Резервное копирование баз данных`" - `Росторгуев Никита`


### Задание 1. Резервное копирование  

1. Для восстановления данных в полном объеме за предыдущий день достаточно делать ежедневное полное резервное копирование. Предпочтительней делать эту копию в часы минимальной нагрузки, например, ночью. Хранить бекапы за 2-3 последних дня и проверять их целостность  
2. Для восстановления данных за час до преполагаемой поломки можно делать полное резервное копирование каждый час, но этот процесс может мешать работе пользователей. Поэтому будет правильней, например, делать полную резервную копию раз в сутки, и далее каждый час делать дифференциальную или инкерементную копию
